# 📱 AI-Powered IoT CCTV Using Old Smartphone

## 📌 Overview
This project converts an unused smartphone into a smart IoT-based CCTV surveillance system. It provides:
- Live video streaming
- Motion detection
- AI-based monitoring using computer vision

Users can remotely monitor their environment and receive real-time alerts.

---

## 🎯 Objectives
- Reuse old smartphone as a surveillance camera  
- Enable real-time remote monitoring  
- Detect motion and suspicious activity  
- Implement AI-based detection using computer vision  
- Send alerts to users  

---

## 🧰 Tech Stack
- **Mobile App:** IP Webcam / DroidCam  
- **Programming Language:** Python  
- **Library:** OpenCV  
- **Cloud Platform:** Firebase (optional)  
- **Alert System:** Telegram Bot / Email  
- **Frontend (optional):** Flask  

---

## 🏗️ System Architecture
Old Smartphone Camera
        ↓
Streaming App (IP Webcam)
        ↓
Local Server (Python)
        ↓
AI Processing (OpenCV)
        ↓
Cloud Storage (Firebase)
        ↓
User Device (Mobile / Laptop)

## ⚙️ Features
- Live video streaming  
- Motion detection  
- Face detection (AI-based)  
- Automatic image capture  
- Alert notifications (Telegram/Email)  
- Cloud storage support  
- Remote access
